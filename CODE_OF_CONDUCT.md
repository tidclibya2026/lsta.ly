# مدونة السلوك — LSTA

تهدف منصة أطلس ليبيا السياحي الذكي إلى دعم عمل حكومي مهني قائم على الاحترام والدقة والمسؤولية.

## السلوك المتوقع

- احترام جميع أعضاء الفريق والشركاء.
- مناقشة القرارات الفنية بالأدلة والاختبارات.
- حماية البيانات الحكومية والسرية المهنية.
- توثيق القرارات والتغييرات بوضوح.
- قبول المراجعة وتصحيح الأخطاء دون شخصنة.
- تجنب تضارب المصالح والإفصاح عنه عند وجوده.
- الالتزام بنطاق المهمة وعدم إدخال تغييرات غير متفق عليها.

## السلوك غير المقبول

- الإساءة أو التمييز أو الترهيب.
- نشر بيانات أو أسرار أو ملفات غير مصرح بها.
- تجاوز دورة المراجعة والاعتماد.
- تغيير النتائج أو البيانات دون سجل تدقيق.
- تعطيل الاختبارات أو الضوابط لتسريع الدمج.
- استخدام المشروع لأغراض شخصية أو تجارية دون تفويض.

## الإنفاذ

تُرفع المخالفات عبر القنوات الإدارية الرسمية للمشروع. ويجوز اتخاذ إجراءات تشمل التنبيه، تقييد الصلاحيات، رفض المساهمة، أو الإحالة إلى الإدارة المختصة.

تنطبق هذه المدونة على المستودع، الاجتماعات، المراسلات، مراجعات الكود، والأنشطة المرتبطة بالمشروع.
