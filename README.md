# أطلس ليبيا السياحي الذكي
## Libya Smart Tourism Atlas — LSTA

منصة وطنية حكومية لإدارة وتكامل وتحليل ونشر البيانات الجغرافية السياحية في دولة ليبيا، بإشراف **وزارة السياحة والصناعات التقليدية** وتنفيذ **مركز المعلومات والتوثيق السياحي**.

> **الشعار المؤسسي:** من ذاكرة المكان إلى صناعة القرار.

## الغرض

تحويل البيانات السياحية الليبية من ملفات وخرائط ودراسات متفرقة إلى بنية وطنية موحدة تدعم:

- صنع القرار الحكومي.
- التخطيط والتنمية المكانية.
- التنمية السياحية المستدامة.
- الاستثمار السياحي.
- البحث العلمي والتعليم.
- الترويج السياحي وخدمة السائح المحلي والأجنبي.
- التكامل مع منصة **Visit Libya**.

## الفئات المستفيدة

1. صانع القرار الحكومي.
2. الجهات العامة والبلديات.
3. المستثمرون والمطورون السياحيون.
4. الباحثون والطلبة.
5. السائح المحلي.
6. السائح الأجنبي.
7. شركاء البيانات والمؤسسات الوطنية.

## المكونات الرئيسية

- الأطلس الوطني التفاعلي.
- بوابة دوائر ومناطق التنمية السياحية.
- محرك التحليل المكاني.
- لوحة مؤشرات الاستدامة والجاهزية.
- الخارطة الاستثمارية السياحية.
- المستودع الوطني للدراسات والتقارير.
- المساعد الافتراضي الذكي متعدد الأدوار.
- واجهات التكامل مع Visit Libya والأنظمة الحكومية.

## المبادئ الحاكمة

- مصدر موثوق لكل بيان.
- معرف وطني موحد لكل موقع.
- سجل تاريخي كامل للتعديلات.
- فصل الإدخال والمراجعة والاعتماد والنشر.
- عدم نشر أي معلومة غير معتمدة.
- دعم العربية والإنجليزية.
- حماية البيانات والأمن السيبراني منذ التصميم.
- الالتزام بمعايير GIS والبيانات الوصفية والتشغيل البيني.

## الحالة الحالية

يحتوي الفرع `main` على النسخة التشغيلية الحالية للخريطة التفاعلية. يجري تطوير النسخة المؤسسية الجديدة تدريجيًا عبر الفروع وطلبات السحب دون تعطيل النسخة القائمة.

## مسار التطوير

- `main`: النسخة المستقرة المنشورة.
- `foundation/government-platform-v1`: التأسيس المؤسسي والتقني.
- فروع المزايا: لكل مكون أو وحدة عمل.
- الدمج يتم فقط عبر Pull Request بعد المراجعة.
