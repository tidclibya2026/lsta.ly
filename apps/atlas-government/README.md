# الواجهة الحكومية لمنصة أطلس ليبيا السياحي الذكي

تطبيق Next.js وTypeScript مخصص للوحة الوزير وصناع القرار، ويعمل بالعربية من اليمين إلى اليسار.

## التشغيل المحلي

```bash
cd apps/atlas-government
npm install
npm run dev
```

ثم افتح `http://localhost:3000`.

## فحوصات الجودة

```bash
npm run typecheck
npm run build
```

## حالة البيانات

القيم الحالية مزيج من بيانات عمل معروفة وقيم نموذجية لتثبيت تجربة الاستخدام. لا تعد مصدرًا رسميًا للنشر العام قبل ربط التطبيق بخدمات LSTA API وقاعدة PostgreSQL/PostGIS.

## المرحلة التالية

1. إنشاء طبقة API موحدة للمؤشرات والتنبيهات.
2. ربط الخريطة بخدمة GeoJSON أو OGC API Features.
3. إضافة المصادقة المؤسسية والصلاحيات.
4. نقل مكونات التصميم إلى حزمة مشتركة داخل Monorepo.
5. إضافة اختبارات واجهة وإمكانية وصول وتدقيق أمني.
