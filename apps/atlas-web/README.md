# لوحة الوزير التنفيذية — LSTA Government UI

نموذج واجهة حكومية أولي لمنصة أطلس ليبيا السياحي الذكي، مبني بـ HTML وCSS وJavaScript ويعمل دون خطوة بناء.

## التشغيل المحلي

```bash
cd apps/atlas-web
python -m http.server 8080
```

ثم افتح:

```text
http://localhost:8080
```

## ما يتضمنه الإصدار 1.0

- اتجاه عربي RTL كامل.
- هوية بصرية حكومية بالأزرق البحري والذهبي.
- لوحة مؤشرات تنفيذية.
- خريطة وطنية تفاعلية عبر Leaflet.
- أوضاع عرض للمواقع والجودة والاستثمار.
- مركز تنبيهات وأولويات متابعة.
- رسوم توزيع مبسطة دون مكتبات رسوم إضافية.
- مساعد وزير تجريبي قابل للربط لاحقًا بخدمة الذكاء الاصطناعي.
- بحث مكاني أولي.
- مظهر فاتح وداكن.
- طباعة تنفيذية واستجابة للهواتف والأجهزة اللوحية.

## تنبيه البيانات

الأرقام داخل الواجهة تجمع بين أرقام المشروع المعروفة وقيم عرض نموذجية. يجب عدم اعتبار الواجهة مصدرًا رسميًا قبل ربطها بقاعدة PostgreSQL/PostGIS وواجهات LSTA API واعتماد مؤشرات الإصدار.
