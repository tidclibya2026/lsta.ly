# إطار حوكمة بيانات أطلس ليبيا السياحي الذكي

## 1. الهدف

ضمان أن تكون بيانات المنصة دقيقة، موثوقة، قابلة للتتبع، آمنة، ومناسبة للاستخدام الحكومي والتخطيطي والاستثماري والبحثي والترويجي.

## 2. الأدوار

- **مالك البيانات:** الجهة المختصة قانونيًا أو مؤسسيًا بالبيان.
- **أمين البيانات:** المسؤول عن إدارة الحقول والتصنيفات والجودة.
- **مدخل البيانات:** يجمع ويدخل البيانات والمصادر.
- **المراجع الفني:** يتحقق من البنية والإحداثيات والتصنيف.
- **المراجع الموضوعي:** يتحقق من صحة المحتوى السياحي.
- **جهة الاعتماد:** تعتمد البيان للنشر المؤسسي.
- **مسؤول النشر:** يدير إتاحة البيان للجمهور أو للشركاء.

## 3. دورة حياة البيان

```text
جمع → إدخال → تحقق آلي → مراجعة فنية → مراجعة موضوعية → اعتماد → نشر → تحديث أو أرشفة
```

## 4. حالات السجل

- مسودة.
- قيد المراجعة.
- يحتاج تحقق ميداني.
- معتمد داخليًا.
- معتمد للنشر.
- موقوف مؤقتًا.
- مؤرشف.

## 5. الحقول الإلزامية لكل موقع

- المعرف الوطني الموحد.
- الاسم العربي.
- الاسم الإنجليزي عند توفره.
- التصنيف الرئيسي والفرعي.
- الإحداثيات ونظام الإسناد.
- البلدية والوحدة الإدارية.
- مصدر البيان.
- الجهة المالكة.
- تاريخ الجمع والتحديث.
- حالة التحقق.
- مستوى الإتاحة.
- وصف مختصر.
- سجل التعديلات.

## 6. الجودة

تقاس الجودة وفق الأبعاد التالية:

- الاكتمال.
- الدقة المكانية.
- الدقة الموضوعية.
- الاتساق.
- الحداثة.
- عدم الازدواجية.
- قابلية التتبع.
- الملاءمة للاستخدام.

## 7. منع الازدواجية

يتم الفحص عبر مجموعة قواعد:

- التشابه النصي للاسم.
- القرب المكاني ضمن نطاق محدد.
- تطابق التصنيف والبلدية.
- تطابق معرفات المصادر.
- المراجعة البشرية للحالات المشتبهة.

لا يحذف سجل مشتبه تلقائيًا؛ بل يوضع في قائمة مراجعة مع اقتراح دمج أو فصل.

## 8. مستويات الإتاحة

- عام.
- حكومي داخلي.
- مقيد حسب الجهة.
- سري أو حساس.

يمنع نشر الإحداثيات أو الوثائق الحساسة للمواقع المعرضة للمخاطر دون موافقة.

## 9. السجل التاريخي

كل تعديل يجب أن يسجل:

- المستخدم.
- التاريخ والوقت.
- القيمة السابقة.
- القيمة الجديدة.
- سبب التعديل.
- المستند أو المصدر الداعم.
- حالة الاعتماد.

## 10. إدارة المصادر

كل معلومة يجب أن ترتبط بمصدر محدد، مثل:

- دراسة معتمدة.
- كتاب أو خريطة.
- جهة حكومية.
- مسح ميداني.
- صورة موثقة.
- قاعدة بيانات رسمية.

ويحفظ اسم المصدر، الجهة، التاريخ، رقم الإصدار، والرابط أو المرفق.

## 11. مراجعة دورية

- المواقع الحيوية والخدمات: مراجعة سنوية أو عند التغيير.
- المواقع الأثرية والطبيعية: مراجعة دورية وميدانية حسب المخاطر.
- الفرص الاستثمارية: مراجعة ربع سنوية أو عند تغير الوضع القانوني.
- المؤشرات: حسب دورية المؤشر.
- الوثائق والدراسات: عند صدور إصدار أحدث.

## 12. الذكاء الاصطناعي

لا تستخدم بيانات غير معتمدة في الإجابات المؤسسية للمساعد الذكي. يجب إظهار المصدر، وتاريخ تحديثه، والتنبيه عند انخفاض الثقة أو تعارض المصادر.
