# ميثاق مشروع أطلس ليبيا السياحي الذكي

## 1. اسم المشروع

**أطلس ليبيا السياحي الذكي — Libya Smart Tourism Atlas (LSTA)**

## 2. الجهة المالكة

وزارة السياحة والصناعات التقليدية — دولة ليبيا.

## 3. الجهة المنفذة

مركز المعلومات والتوثيق السياحي.

## 4. خلفية المشروع

تتوزع البيانات والمعلومات السياحية بين خرائط ودراسات وتقارير وصور وقواعد بيانات وملفات مؤسسية متعددة، بما يحد من قدرتها على خدمة التخطيط والاستثمار وصناعة القرار. يأتي المشروع لبناء منصة وطنية موحدة تربط البيانات السياحية بالمكان والزمن والمصدر وحالة الاعتماد.

## 5. الرؤية

بناء منظومة وطنية ذكية تجعل البيانات السياحية الجغرافية مصدرًا موثوقًا لصناعة القرار وقيادة التنمية السياحية المستدامة في ليبيا.

## 6. الرسالة

جمع وتصنيف وتوثيق وربط وتحليل ونشر البيانات السياحية وفق حوكمة مؤسسية ومعايير تقنية ومكانية دولية، وتحويلها إلى خرائط ومؤشرات وتقارير وخدمات رقمية تدعم الدولة والمستثمر والباحث والسائح.

## 7. الأهداف الاستراتيجية

1. إنشاء قاعدة بيانات جغرافية سياحية وطنية موحدة.
2. توحيد منهجيات جمع وتصنيف والتحقق من البيانات.
3. إدماج نتائج دراسات دوائر التنمية السياحية السابقة.
4. دعم التخطيط والتنمية المكانية المتوازنة.
5. إدماج مؤشرات الاستدامة البيئية والاجتماعية والاقتصادية والمؤسسية.
6. دعم الاستثمار وتحديد الأولويات والفرص.
7. إتاحة المعلومات الموثوقة لصانع القرار والباحث والطالب.
8. تقديم محتوى وخدمات ذكية للسائح المحلي والأجنبي.
9. بناء مساعد افتراضي يعتمد حصريًا على المصادر المعتمدة.
10. التكامل المنظم مع Visit Libya والأنظمة الحكومية ذات الصلة.

## 8. النطاق

### داخل النطاق

- المواقع والمقومات والمرافق والخدمات السياحية.
- دوائر ومناطق التنمية السياحية.
- الدراسات والتقارير والخرائط التاريخية والحديثة.
- المؤشرات السياحية والتنموية والاستدامية.
- الفرص والمشروعات الاستثمارية.
- التحليل المكاني والسيناريوهات.
- إدارة المصادر والتحقق وسجل التغييرات.
- واجهات API والتكامل مع Visit Libya.
- المساعد الافتراضي الذكي.

### خارج النطاق في الإصدار الأول

- المعاملات المالية الحكومية.
- منح التراخيص القانونية بصورة نهائية.
- أنظمة الحجز والدفع التجاري المباشر.
- نشر البيانات الحساسة أو المقيدة دون موافقة.

## 9. المخرجات الرئيسية

- قاعدة PostgreSQL/PostGIS وطنية.
- بوابة إدارة مؤسسية متعددة الصلاحيات.
- أطلس تفاعلي ثنائي اللغة.
- لوحة دعم القرار.
- بوابة دوائر التنمية السياحية.
- مؤشر الاستدامة والجاهزية.
- مكتبة الدراسات والوثائق.
- API موثقة.
- مساعد افتراضي قائم على RAG مع إظهار المصادر.
- دليل حوكمة البيانات والتشغيل.

## 10. أصحاب المصلحة

- وزير السياحة والصناعات التقليدية.
- مركز المعلومات والتوثيق السياحي.
- الإدارات والجهات التابعة للوزارة.
- البلديات والجهات الحكومية الشريكة.
- جهاز الآثار والشرطة السياحية والطيران والجوازات والإحصاء والمعلومات.
- الجامعات ومراكز البحث.
- المستثمرون والقطاع الخاص.
- المجتمع المحلي والسياح.

## 11. مبادئ القبول

- لكل موقع معرف وطني ومصدر وحالة تحقق وتاريخ تحديث.
- العربية والإنجليزية تعملان في الواجهة والمحتوى الأساسي.
- الخرائط تدعم آلاف النقاط دون تدهور غير مقبول.
- الفصل بين البيانات المؤسسية والمحتوى الترويجي.
- جميع التعديلات الحساسة قابلة للتتبع.
- لا يقدم المساعد الذكي إجابة مؤسسية بلا مصدر.
- التكامل مع Visit Libya يتم عبر API وليس النسخ اليدوي.

## 12. النجاح المؤسسي

يعد المشروع ناجحًا عند اعتماد المنصة كمصدر مرجعي موحد للبيانات الجغرافية السياحية، واستخدام مخرجاتها دوريًا في التقارير الحكومية والتخطيط والاستثمار والترويج والبحث العلمي.

## 13. الاعتماد

**إعداد:** مركز المعلومات والتوثيق السياحي  
**الإشراف:** وزارة السياحة والصناعات التقليدية  
**الإصدار:** 1.0 — 2026
