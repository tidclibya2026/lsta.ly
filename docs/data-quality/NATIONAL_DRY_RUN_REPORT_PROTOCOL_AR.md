# بروتوكول التقرير الوطني للفحص التجريبي لبيانات الأطلس

## 1. الغرض

توحيد طريقة تشغيل الفحص التجريبي على طبقات أطلس ليبيا السياحي، وتوثيق النتائج، وتصنيف المشكلات، وتحويلها إلى قوائم عمل قابلة للتنفيذ من فريق البيانات ومختصي GIS وجهات الاعتماد.

## 2. نطاق التقرير

يشمل التقرير الطبقات الأساسية التالية:

1. الفنادق.
2. القرى والمنتجعات السياحية.
3. المطاعم.
4. المقاهي.
5. الطبقات الموضوعية المنفصلة حسب المحاور.

لا يعد الفحص التجريبي اعتمادًا نهائيًا للبيانات، ولا يسمح بالنشر العام أو الإرسال إلى Visit Libya.

## 3. المخرجات الإلزامية

- ملخص تنفيذي لكل طبقة.
- عدد السجلات المقروءة.
- عدد مشكلات الجودة المكتشفة.
- بصمة SHA-256 لكل ملف.
- قائمة الملفات أو الأوراق التي تعذر قراءتها.
- قائمة السجلات التي تحتاج تحققًا ميدانيًا.
- قائمة حالات الاشتباه في التكرار أو الدمج.
- قائمة السجلات المرشحة للاعتماد المكتبي.
- توصيات المعالجة حسب الأولوية والجهة المسؤولة.

## 4. تصنيف الأولوية

### أولوية حرجة

- إحداثيات غير صالحة أو خارج نطاق ليبيا.
- سجل بلا اسم.
- تعارض جوهري بين المصدر والموقع.
- ملف تالف أو ورقة غير قابلة للقراءة.

### أولوية عالية

- مصدر غير موثق.
- معرف مفقود أو متكرر.
- اشتباه قوي في التكرار المكاني والنصي.
- حالة تحقق غير محددة.

### أولوية متوسطة

- بلدية غير محددة.
- اسم إنجليزي أو وصف أو وسيلة اتصال مفقودة.
- اختلاف تنسيقي يمكن تصحيحه مكتبيًا.

### أولوية منخفضة

- تحسينات تحريرية أو صور إضافية.
- حقول اختيارية غير مكتملة.

## 5. قواعد القرار

- لا يحذف أي سجل أثناء الفحص التجريبي.
- لا يدمج أي سجل آليًا.
- تحفظ النسخة الأصلية للصف والملف وبصمته.
- يعالج الاشتباه في التكرار عبر مراجعة بشرية موثقة.
- لا ينتقل السجل إلى Visit Libya قبل استكمال بوابات الهوية والمكان والمصدر والتحقق والنشر.

## 6. أمر التشغيل

```bash
python tools/run_dry_run_batch.py \
  --input-dir data/raw/atlas-2026 \
  --output-dir reports/generated
```

ينتج الأمر:

- `atlas_dry_run_summary.json`
- `atlas_dry_run_summary.csv`

## 7. اعتماد التقرير

يمر التقرير بالمراجعات التالية:

1. مسؤول البيانات: سلامة الاستخراج والحقول.
2. مختص GIS: الإحداثيات والإسناد المكاني والبلديات.
3. مسؤول الجودة: الأولويات والتكرار والمصادر.
4. مدير المشروع: اعتماد خطة المعالجة.
5. الجهة المخولة: اعتماد البيانات للنشر أو الإرجاع للمراجعة.
