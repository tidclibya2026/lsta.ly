# LSTA Government Alpha v0.1.0

أول إصدار حكومي تشغيلي لمنصة أطلس ليبيا السياحي الذكي.

## أبرز القدرات

- بوابة حكومية عربية RTL مبنية بـ Next.js وTypeScript.
- خريطة MapLibre تفاعلية وطبقات مراجعة مكانية.
- مستورد KML/KMZ مع تقارير جودة وبصمات SHA-256.
- PostgreSQL/PostGIS وبيئة Docker محلية.
- قاعدة Staging منفصلة عن السجل الوطني.
- بوابة مراجعة واعتماد متعددة المراحل.
- محرك جودة بيانات وصلاحيات حكومية أولية.
- سجل وطني وSite Profile وإصدارات وتدقيق.
- محرك علاقات مكانية ومرشحو Nearby عبر PostGIS.

## حالة البيانات عند نقطة الإصدار

- عناصر Staging: 430.
- المواقع الوطنية: 1.
- Promotions: 1.
- Site Profiles: 1.
- العلاقات المكانية التجريبية: 10، جميعها قيد المراجعة.

## الضوابط

- هذا إصدار Alpha للاختبار والمراجعة الداخلية.
- لا يتضمن نشرًا إلى Visit Libya.
- لا يتضمن أسرارًا أو بيانات خامًا أو وسائط غير معتمدة.
- لا تعتبر البيانات التجريبية منشورة أو معتمدة للعامة.

## التحقق المطلوب قبل إنشاء Release

- نجاح Government Portal tests وtypecheck وbuild.
- نجاح Backend tests وRuff وAlembic migrations.
- نجاح اختبارات KML Importer.
- مراجعة عدم وجود ملفات `.env` أو بيانات خام في التغييرات.
- اعتماد Pull Request الخاص بالإصدار.
