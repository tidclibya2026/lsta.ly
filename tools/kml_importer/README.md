# مستورد KML وKMZ — منصة أطلس ليبيا السياحي الذكي LSTA

أداة مستقلة وآمنة لتحويل ملفات Google My Maps إلى مخرجات مرحلية قابلة للمراجعة. لا تعدّل ملفات المصدر، ولا تتصل بـPostgreSQL أو أي بيئة إنتاج.

## التشغيل

من جذر المستودع:

```bash
python -m tools.kml_importer import --input data/raw/example.kml --output data/processed/kml --report-dir reports/kml-import
```

في المرحلة التجريبية الحالية يكون `--input` ملف KML. ما زالت الصيغة السابقة من دون كلمة `import` وخيار `--report` المباشر مدعومة للتوافق.

## المخرجات

- `features.geojson`: العناصر والهندسات وخصائصها.
- `features.csv`: نسخة مسطحة UTF-8 BOM ملائمة للمراجعة المكتبية.
- `import.json`: النموذج الكامل، بما فيه الملفات وبصمات SHA-256.
- `quality.json`: مؤشرات الجودة بصيغة آلية.
- `quality-report.md`: تقرير عربي قابل للقراءة.
- `assets/`: الصور المضمنة المستخرجة من KMZ.

يحتفظ النموذج بـHTML الوصف الأصلي والنص النظيف وجداول Key/Value والحقول الممتدة والحقول غير المعروفة. تُسجّل الإحداثيات غير الصالحة والصور المفقودة بدل إسقاط العنصر كاملًا.

## الخصوصية

تمنع قواعد `.gitignore` رفع KML وKMZ والصور الخام تحت `data/raw/`. راجع المخرجات المنقّحة قبل مشاركتها أو اعتمادها؛ هذه المرحلة لا تنشر البيانات ولا تكتبها إلى قاعدة البيانات.
